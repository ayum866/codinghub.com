13 Cool CSS Animation Examples

1. Simple Fade In/Out

/* CSS for Fade In/Out */
@keyframes fadeInOut {
    0% { opacity: 0; }
    50% { opacity: 1; }
    100% { opacity: 0; }
}
.fade-box {
    animation: fadeInOut 3s infinite alternate;
}
Fade

Fades an element in and out smoothly.

2. Slide In from Left

/* CSS for Slide In from Left */
@keyframes slideInLeft {
    0% { transform: translateX(-100%); opacity: 0; }
    100% { transform: translateX(0); opacity: 1; }
}
.slide-box {
    animation: slideInLeft 1.5s ease-out forwards;
}
Slide

Slides an element into view from the left.

3. Pulse Effect

/* CSS for Pulse Effect */
@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}
.pulse-box {
    animation: pulse 2s infinite ease-in-out;
}
Pulse

Makes an element gently grow and shrink, simulating a pulse.

4. Rotate 360 Degrees

/* CSS for Rotate 360 Degrees */
@keyframes rotate360 {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
.rotate-box {
    animation: rotate360 4s linear infinite;
}
Rotate

Continuously rotates an element around its center.

5. Bounce Effect

/* CSS for Bounce Effect */
@keyframes bounce {
    0%, 20%, 50%, 80%, 100% { transform: translateY(0); }
    40% { transform: translateY(-30px); }
    60% { transform: translateY(-15px); }
}
.bounce-box {
    animation: bounce 2s infinite;
}
Bounce

Simulates a bouncing motion for an element.

6. Color Change

/* CSS for Color Change */
@keyframes colorChange {
    0% { background-color: #3498db; }

    50% { background-color: #2ecc71; }
    75% { background-color: #f1c40f; }
    100% { background-color: #3498db; }

.color-box {
    animation: colorChange 5s infinite;

Color

Transitions an element's background color through a sequence.

7. Grow/Shrink & Morph

/* CSS for Grow/Shrink & Morph */
@keyframes growShrink {
    0% { transform: scale(1); }
    50% { transform: scale(1.2); border-radius: 50%; }
    100% { transform: scale(1); border-radius: 8px; }
}
.grow-box {
    animation: growShrink 3s infinite ease-in-out;
}
Grow

Makes an element grow, shrink, and subtly morph its shape.

Skew

9. Text Typing Effect

/* CSS for Text Typing Effect */
.typing-text {
    font-family: 'Courier New', Courier, monospace;
    font-size: 1.2em;
    color: #2c3e50;
    overflow: hidden;
    border-right: .15em solid orange;
    white-space: nowrap;
    margin: 0 auto;
    letter-spacing: .15em;
    animation: typing 3.5s steps(30, end) infinite,
               blink-caret .75s step-end infinite;
}
@keyframes typing {
    from { width: 0 }
    to { width: 100% }
}
@keyframes blink-caret {
    from, to { border-color: transparent }
    50% { border-color: orange; }
}
Hello, World!

Creates a classic typewriter effect for text.

10. Gradient Background Animation

/* CSS for Gradient Background Animation */
@keyframes gradientShift {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}
.gradient-box {
    width: 150px;
    height: 150px;
    background: linear-gradient(270deg, #ff9a9e, #fad0c4, #fad0c4, #ff9a9e);
    background-size: 400% 400%;
    animation: gradientShift 10s ease infinite;
}
Gradient

Animates a multi-color gradient background.

11. Shake Effect

/* CSS for Shake Effect */
@keyframes shake {
    0%, 100% { transform: translateX(0); }
    10%, 30%, 50%, 70%, 90% { transform: translateX(-5px); }
    20%, 40%, 60%, 80% { transform: translateX(5px); }
}
.shake-box {
    animation: shake 0.8s cubic-bezier(.36,.07,.19,.97) infinite;
}
Shake

Applies a subtle horizontal shaking motion.

12. Fade In Up

/* CSS for Fade In Up */
@keyframes fadeInUp {

    100% { opacity: 1; transform: translateY(0); }
}
.fade-in-up-box {

}
Appear

Fades an element in while moving it slightly upwards.

13. Continuous Flip

/* CSS for Continuous Flip */
@keyframes flip {
    0% { transform: rotateY(0deg); }
    100% { transform: rotateY(360deg); }
}
.flip-box {
    animation: flip 4s linear infinite;
    transform-style: preserve-3d;
}
Flip

Rotates an element continuously on its Y-axis, creating a flip effect.
