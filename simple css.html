More Simple CSS Examples

  1. Rounded Corners (border-radius)

    The border-radius property is used to add rounded corners to an element's border.

    Example HTML:

    <div class="rounded-box">This box has rounded corners.</div>
<button class="rounded-button">Rounded Button</button>

    Example CSS:

    
.rounded-box {
    background-color: #abebc6; /* Light green */
    padding: 20px;
    border: 2px solid #58d68d; /* Green */
    border-radius: 15px; /* You can use px, %, or em */
    text-align: center;
}
.rounded-button {
    background-color: #85c1e9; /* Light blue */
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 25px; /* Makes it pill-shaped if height allows */
    cursor: pointer;
    margin-top: 10px;
}

    Result Preview:

    This box has rounded corners.
  2. Drop Shadows (box-shadow)

    The box-shadow property adds shadow effects around an element's frame. You can specify multiple shadows.

    Example HTML:

    <div class="shadow-box">This box has a subtle shadow.</div>

    Example CSS:

    
.shadow-box {
    background-color: #fdebd0; /* Pale orange */
    padding: 20px;
    border-radius: 5px;
    /* box-shadow: h-offset v-offset blur spread color inset; */
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
    text-align: center;
}

    Result Preview:

    This box has a subtle shadow.
  3. Transparency (opacity)

    The opacity property specifies the transparency of an element. The value ranges from 0.0 (fully transparent) to 1.0 (fully opaque).

    Example HTML:

    <div class="opaque-box">Fully Opaque</div>
<div class="semi-transparent-box">Semi-Transparent (0.6)</div>

    Example CSS:

    
.opaque-box, .semi-transparent-box {
    width: 150px;
    padding: 15px;
    margin-bottom: 10px;
    color: white;
    text-align: center;
}
.opaque-box {
    background-color: #c0392b; /* Pomegranate */
    opacity: 1.0; /* Default */
}
.semi-transparent-box {
    background-color: #2980b9; /* Belize Hole */
    opacity: 0.6;
}

    Result Preview:

    Fully Opaque
    Semi-Transparent (0.6)
  4. Styling Lists (list-style-type)

    The list-style-type property changes the appearance of the list item marker (e.g., bullets, numbers).

    Example HTML:

    <ul class="custom-list-square">
    <li>Item 1 (Square)</li>
    <li>Item 2 (Square)</li>
</ul>
<ul class="custom-list-none">
    <li>Item A (No marker)</li>
    <li>Item B (No marker)</li>
</ul>
<ol class="custom-list-roman">
    <li>First (Upper Roman)</li>
    <li>Second (Upper Roman)</li>
</ol>

    Example CSS:

    
.custom-list-square {
    list-style-type: square;
}
.custom-list-none {
    list-style-type: none; /* Removes markers */
    padding-left: 0; /* Often desired when removing markers */
}
.custom-list-roman {
    list-style-type: upper-roman;
}

    Result Preview:

    • Item 1 (Square)
    • Item 2 (Square)
    • Item A (No marker)
    • Item B (No marker)
    1. First (Upper Roman)
    2. Second (Upper Roman)
  5. Changing Mouse Cursor (cursor)

    The cursor property specifies the mouse cursor to be displayed when pointing over an element.

    Example HTML:

    <p class="cursor-pointer">Hover here for a pointer cursor.</p>
<p class="cursor-help">Hover here for a help cursor.</p>
<p class="cursor-wait">Hover here for a wait cursor.</p>

    Example CSS:

    
.cursor-pointer {
    cursor: pointer; /* Common for clickable items */
    background-color: #f2f3f4; padding: 5px;
}
.cursor-help {
    cursor: help;
    background-color: #f2f3f4; padding: 5px;
}
.cursor-wait {
    cursor: wait;
    background-color: #f2f3f4; padding: 5px;
}

    Result Preview:

    Hover here for a pointer cursor.

    Hover here for a help cursor.

    Hover here for a wait cursor.

  6. Handling Content Overflow (overflow)

    The overflow property controls what happens to content that is too big to fit into an element's box.

    • visible: Default. Content renders outside the element's box.
    • hidden: Content is clipped and the rest is invisible.
    • scroll: Content is clipped, but scrollbars are added to see the rest.
    • auto: Adds scrollbars only when necessary.

    Example HTML:

    <div class="overflow-box scroll-example">
    This is a lot of text that will likely overflow this small box.
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
    Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</div>
<div class="overflow-box hidden-example">
    This text will be hidden if it overflows. Lorem ipsum dolor sit amet, consectetur adipiscing elit.
</div>

    Example CSS:

    
.overflow-box {
    width: 200px;
    height: 70px;
    border: 1px solid #7d3c98; /* Purple */
    padding: 10px;
    margin-bottom: 15px;
    background-color: #e8daef; /* Light purple */
}
.scroll-example {
    overflow: scroll; /* Always show scrollbars */
}
.hidden-example {
    overflow: hidden; /* Clip content */
}

    Result Preview:

    This is a lot of text that will likely overflow this small box. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
    This text will be hidden if it overflows. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  7. Hiding Elements (display: none vs visibility: hidden)

    Two common ways to hide elements:

    • display: none;: Removes the element from the document flow entirely. It takes up no space.
    • visibility: hidden;: Hides the element, but it still occupies its space in the layout.

    Example HTML:

    <div>Element Before</div>
<div class="hidden-display-none">Hidden with display: none</div>
<div>Element After (display: none)</div>
<hr>
<div>Element Before</div>
<div class="hidden-visibility">Hidden with visibility: hidden</div>
<div>Element After (visibility: hidden)</div>

    Example CSS:

    
.hidden-display-none, .hidden-visibility, div {
    padding: 10px;
    border: 1px solid #ccc;
    margin: 5px;
    background-color: #f9e79f; /* Light yellow */
}
.hidden-display-none {
    display: none;
}
.hidden-visibility {
    visibility: hidden;
}

    Result Preview:

    Element Before
    Hidden with display: none
    Element After (display: none)
    Element Before
    Hidden with visibility: hidden
    Element After (visibility: hidden)
