10 Simple JavaScript Examples

1. Hello World (Console Log)

Open your browser's developer console (F12 or right-click -> Inspect, then go to the Console tab) to see the output.

console.log("Hello, World!");

2. Changing HTML Content

This text will change when the button is clicked.

document.getElementById("changeText").innerHTML = "Text has been changed by JavaScript!";

3. Simple Alert Box

Click the button to see a simple alert message.

alert("This is a simple alert!");

4. Basic Arithmetic

Demonstrates adding two numbers.

let a = 5;
let b = 10;
let sum = a + b;
document.getElementById("output4").innerText = "The sum of 5 and 10 is: " + sum;

5. Function Definition and Call

A function that returns a greeting.

function greet(name) {
    return "Hello, " + name + "!";
}
document.getElementById("output5").innerText = greet("Alice");

6. Conditional Statement (if-else)

Checks if a number is even or odd.

let num = 7;
if (num % 2 === 0) {
    document.getElementById("output6").innerText = num + " is an even number.";
} else {
    document.getElementById("output6").innerText = num + " is an odd number.";
}

7. Loop (for loop)

Counts from 1 to 5.

let loopOutput = "Counting: ";
for (let i = 1; i <= 5; i++) {
    loopOutput += i + " ";
}
document.getElementById("output7").innerText = loopOutput;

8. Array and Accessing Elements

Creates an array of fruits and displays the second one.

let fruits = ["Apple", "Banana", "Cherry"];
document.getElementById("output8").innerText = "My favorite fruit is: " + fruits[1]; // Banana

9. Event Listener

Click the button below to see an output from an event listener.

document.getElementById("eventButton").addEventListener("click", function() {
    document.getElementById("output9").innerText = "Button clicked via event listener!";
});

10. Date Object

Displays the current date and time.

let currentDate = new Date();
document.getElementById("output10").innerText = "Current Date and Time: " + currentDate.toLocaleString();
